## Stater KIT - Laravel MQTT Gateway

Project dasar siap di pakai dengan konfigurasi dasar yang sudah di sesidakan.

Konfigurasi:

- Konfigurasi Server/Broker mqtt pada config -> mqtt.php
- Konfigruasi bisa di lakukan melalui .env -> tersedia di .env.example
- Mqtt Service sudah tersedia dan sudah di optimasi dengan ringan.
